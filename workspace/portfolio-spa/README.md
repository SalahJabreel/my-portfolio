# Portfolio SPA

Single Page Application portfolio for صلاح الدين جبريل built with React + Vite + Tailwind CSS + Framer Motion.

## Development

- Install deps: `npm install`
- Run dev server: `npm run dev`

## EmailJS setup

Create a `.env` file in the project root with:

```
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

Update `public/cv.pdf` with your actual CV and `public/profile-placeholder.png` with your photo.
