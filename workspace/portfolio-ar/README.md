## Environment
Create `.env` in project root with EmailJS keys:
```
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

## Notes
- RTL and Arabic font (Cairo) are enabled in `index.html`.
- Tailwind is configured in `tailwind.config.js` and `postcss.config.js`.
- Profile image placeholder at `src/assets/profile.svg`.
